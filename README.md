# ColourBridge

**Color-Blind Companion for iOS**

An offline accessibility tool that helps people with color-vision deficiencies interpret color-coded information (charts, maps, labels) using the camera and image analysis.

## Features

- **Live Camera Analysis**: Point your camera at any color-coded information and tap to identify colors in real-time
- **Photo Analysis**: Import photos from your library to analyze color-coded charts, maps, or diagrams
- **Sample Chart Demo**: Try the app with a built-in sample chart to see how it works
- **Color Simulation**: Switch between Normal, Protan, and Deutan vision modes to see how colors appear with different color-vision deficiencies
- **WCAG Contrast Analysis**: Get detailed contrast ratios and readability ratings for each color
- **Smart Color Naming**: Colors are identified with descriptive, accessible names
- **Privacy-First**: All processing happens on-device—no data is uploaded or stored

## Technical Highlights

- Built with SwiftUI and Swift 6
- Core Image for efficient color sampling and transformation
- AVFoundation for real-time camera capture
- Swift Charts for data visualization
- WCAG 2.x contrast ratio calculations
- Color-blind simulation using Viénot/Brettel matrices

## Requirements

- iOS 17.0+
- Camera and Photo Library permissions

## Purpose

Designed for the **Apple Swift Student Challenge 2026** to demonstrate how technology can improve accessibility and help people with color-vision deficiencies navigate a color-coded world.
